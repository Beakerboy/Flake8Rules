---
code: E182
message: "Incorrect comparison symbol format"
title: "Incorrect comparison symbol format (E266)"
---

Do not use the symbols `=>`, `=<`, or `><`. Instead use `>=`, `<=`, or `<>`.

### Anti-pattern

```vba
If Bar >< 4 Then
End If
```

### Best practice

```vba
If Bar <> 4 Then
End If
```
