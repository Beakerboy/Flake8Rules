---
code: E124
message: "Excess whitespace after '('"
title: "Excess whitespace after '(' (E201)"
---

Open parentheses should not have any space after them.

### Anti-pattern

```vba
' The space after Foo is unnecessary
Function Foo( Bar)
End Function
```

### Best practice

```vba
Function Foo(Bar)
End Function
```
