---
code: E120
message: "Missing whitespace bevore '('"
title: "Missing whitespace brfore '(' (E120)"
---

There should be one space before left-parenthesis in define statements.

### Anti-pattern

```vba
Public Define Function Foo "Bar"(Baz)
```

### Best practice

```vba
Public Define Function Foo "Bar" (Baz)
```
